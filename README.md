
## Overview
SweetPastel is simple pastel syntax theme for VSCode.

## Screenshot
![screenshot](https://rawgit.com/imahmoodz/sweetPastel/master/screenshot.png)

## install 
Search "sweetpastel" in extentions and install it.
or run this command
```shell
ext install sweetpastel
```
